# Zenn Articles

[![Zenn](https://img.shields.io/badge/Zenn-Articles-3EA8FF?logo=zenn)](https://zenn.dev/kurozumi)

Zennで公開している記事を管理するリポジトリです。

## 使い方

```bash
# 依存パッケージをインストール
npm install

# プレビュー（localhost:8000）
npm run preview

# 新しい記事を作成
npm run new:article

# 新しい本を作成
npm run new:book
```

## ディレクトリ構成

```
.
├── articles/    # 記事
└── books/       # 本
```
