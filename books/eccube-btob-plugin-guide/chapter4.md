## この章で学ぶこと

BtoBの取引では、同じ商品でも「誰が買うか」によって価格が変わるのが当たり前です。代理店には定価の70%で、卸先には80%で、一般会員には100%（定価）で……といった価格体系を持っている事業者さんは多いのではないでしょうか。

この章では、**価格管理アドオン**を使って、会員グループごとに異なる価格を設定する方法を解説します。コードを1行も書かずに、管理画面の操作だけで卸価格・割引価格を実現できます。

具体的には以下のことを学べます。

- なぜBtoBでは顧客ごとに価格を変える必要があるのか
- 「掛け率」とは何か、どう計算するのか
- 価格管理アドオンのインストールと設定方法
- グループ全体への掛け率設定と、特定商品への個別価格設定の方法
- 端数処理（切り捨て・切り上げ・四捨五入）の設定

---

## なぜグループごとに価格を変える必要があるのか

### BtoBの現実：同じ商品でも値段が違う

BtoC（一般消費者向け）のECサイトでは、基本的に誰が買っても同じ値段です。しかしBtoB（法人・企業向け）の取引では、以下のように**顧客の種類によって価格を変えるのが一般的**です。

| 顧客の種類 | 価格の例 |
|---|---|
| 一般消費者 | 定価（10,000円） |
| 代理店 | 定価の70%（7,000円） |
| 卸先（問屋） | 定価の75%（7,500円） |
| VIP法人会員 | 定価の80%（8,000円） |

これは現実の商取引でごく普通に行われていることです。ECサイトでもこれを実現しないと、「法人取引先に間違えて定価を見せてしまった」「代理店にも一般顧客と同じ価格が表示されてクレームになった」といったトラブルが起きます。

### 「掛け率」とは何か

「掛け率」という言葉になじみのない方のために説明します。

**掛け率とは、定価に対して何割の価格で提供するかを表すパーセンテージのことです。**

たとえば：

- 掛け率80%（0.8掛け） → 定価10,000円の商品が **8,000円**
- 掛け率70%（0.7掛け） → 定価10,000円の商品が **7,000円**
- 掛け率100%（1.0掛け） → 定価10,000円の商品が **10,000円**（変化なし）

「80掛け（はちがけ）」「7掛け（ななかけ）」という言葉を使う業界も多いです。

計算式はシンプルです。

```
販売価格 = 定価 × 掛け率 ÷ 100
```

たとえば定価が 12,500円で掛け率80%の場合：

```
12,500円 × 80 ÷ 100 = 10,000円
```

### 代理店・卸先・VIP会員の具体的な価格体系例

実際のBtoB ECサイトでよく見られる価格体系の例を挙げます。

**製造業A社のケース**

- A社は自社製品をECサイトで販売
- 一般消費者向け（定価）：10,000円
- 認定代理店グループ：7,500円（掛け率75%）
- 特約店グループ：8,500円（掛け率85%）
- 一般法人グループ：9,000円（掛け率90%）

このように複数の価格体系を管理するには、会員をグループに分けてグループごとに掛け率を設定するのがもっとも効率的です。

## 価格管理アドオンの概要

今回使用するのは、EC-CUBEの公式マーケットプレイスで提供されている**価格管理アドオン**です。

- **アドオン名**: 価格管理アドオン
- **URL**: https://www.ec-cube.net/products/detail.php?product_id=2440
- **対応バージョン**: EC-CUBE 4.0〜4.3

### 前提条件：会員グループ管理プラグインが必要

価格管理アドオンを使うには、**第2章で解説した会員グループ管理プラグイン（商品ID: 2439）**が先にインストールされている必要があります。

会員グループ管理プラグインで「法人会員グループ」「代理店グループ」などを作成し、会員をグループに振り分けた状態で、価格管理アドオンを使って各グループの価格を設定します。

会員グループ管理プラグインの設定がまだの方は、第2章に戻って設定を完了させてから、この章に進んでください。

## インストール手順

### 1. アドオンの購入・ダウンロード

1. [価格管理アドオンのページ](https://www.ec-cube.net/products/detail.php?product_id=2440)にアクセスします
2. EC-CUBEオーナーズストアにログインして購入します（有料）
3. 購入後、管理画面の「オーナーズストア」からダウンロードできます

### 2. 管理画面からインストール

1. EC-CUBE管理画面にログインします
2. 左メニューの「オーナーズストア」→「プラグイン」をクリックします
3. ダウンロード済みのプラグイン一覧から「価格管理アドオン」を探します
4. 「インストール」ボタンをクリックします
5. インストール完了後、「有効」に切り替えます

### 3. インストール確認

インストールが完了すると、管理画面の左メニューに「価格管理」という項目が追加されます。また、会員グループの編集画面に「価格設定」タブが追加されているはずです。

## 設定できること

価格管理アドオンでは、主に以下の4つの設定が可能です。

### 1. グループに掛け率を設定する

会員グループ全体に対して掛け率を設定できます。たとえば「法人会員グループ全体に80%掛けを適用する」といった設定が可能です。

グループに掛け率を設定すると、そのグループに属する会員が商品を閲覧したとき、**自動的に掛け率が計算された価格**が表示されます。

### 2. グループに割引率を設定する

掛け率ではなく「割引率」で設定することもできます。

- 掛け率80% = 割引率20%（20%引き）
- 掛け率90% = 割引率10%（10%引き）

どちらの表現で設定するかは、自社の業務慣習に合わせて選べます。

### 3. 商品ごとに個別価格を設定する

グループへの掛け率設定とは別に、**特定の商品だけ個別に価格を設定**することも可能です。

たとえば「法人会員グループには基本80%掛けを適用しているが、この商品だけは特別に7,000円（定価9,000円）にしたい」といった細かい設定ができます。

商品の規格（サイズや色のバリエーション）ごとに個別価格を設定することもできます。

### 4. 端数処理の設定

掛け率を計算すると端数が出ることがあります。たとえば：

```
定価 9,800円 × 80% = 7,840円
定価 1,234円 × 80% = 987.2円
```

小数点以下が出た場合の処理方法を以下から選べます。

- **切り捨て**: 987.2円 → 987円
- **切り上げ**: 987.2円 → 988円
- **四捨五入**: 987.2円 → 987円（0.5未満なので切り捨て）

一般的には「切り捨て」が多く使われますが、業界の慣習や自社のポリシーに合わせて設定してください。

## 具体的な設定例

### 例1：法人会員グループ全体に80%掛けを適用する

最もシンプルな設定例です。法人会員グループに属する会員全員が、すべての商品を定価の80%で購入できるようにします。

**手順**

1. 管理画面の「会員グループ管理」をクリックします
2. 設定したいグループ（例：「法人会員」）の編集ボタンをクリックします
3. 「価格設定」タブをクリックします
4. 「掛け率」の入力欄に `80` と入力します（パーセントなので100未満の数値を入力）
5. 端数処理を選択します（例：「切り捨て」）
6. 「保存」ボタンをクリックします

**確認方法**

設定後、法人会員グループに属するアカウントでサイトにログインして、商品一覧ページを確認します。定価10,000円の商品が8,000円と表示されていれば成功です。

**表示のイメージ**

```
【通常価格】10,000円（税込）
【会員価格】8,000円（税込）
```

商品詳細ページでは、通常価格と会員価格を並べて表示させることもできます（テンプレートの設定による）。

---

### 例2：特定商品だけ法人会員向けに8,000円（通常10,000円）に設定する

グループへの掛け率設定ではなく、特定商品に対して個別価格を設定する例です。

**シナリオ**

- 商品A（定価10,000円）を法人会員グループには **8,000円固定**で販売したい
- 他の商品は通常価格のまま（法人会員グループへの掛け率設定なし）

**手順**

1. 管理画面の「商品管理」→「商品一覧」をクリックします
2. 対象の商品（商品A）の編集ボタンをクリックします
3. 「価格設定」タブまたはセクションをクリックします
4. 「グループ別価格」の設定欄が表示されます
5. 「法人会員グループ」の行に `8000` と入力します
6. 「保存」ボタンをクリックします

**注意点**

グループへの掛け率と商品の個別価格が両方設定されている場合、**商品の個別価格が優先**されます。これにより「グループ全体は80%掛けだが、この商品だけは特別価格」という設定が可能です。

## 価格の適用範囲

価格管理アドオンを設定すると、グループ価格は以下のすべての画面で一貫して適用されます。

| 画面 | 適用 |
|---|---|
| 商品一覧ページ | グループ価格が表示される |
| 商品詳細ページ | グループ価格が表示される |
| カートページ | グループ価格で合計金額が計算される |
| 注文確認ページ | グループ価格で注文内容が確認できる |
| 注文完了後のメール | グループ価格が記載される |

これはとても重要なポイントです。たとえば商品詳細ページだけ安い価格が表示されて、カートに入れると通常価格に戻ってしまうような不整合があると、会員の信頼を損ないます。価格管理アドオンはこの一貫性を担保しています。

## よくある質問

**Q1. ログインしていない（非会員の）ユーザーにはどの価格が表示されますか？**

A. 非会員ユーザーには通常の定価が表示されます。グループ価格はログインしてグループに属している会員にのみ適用されます。

**Q2. 一つの商品に複数のグループ価格を設定できますか？**

A. はい、できます。「法人会員グループ：8,000円」「代理店グループ：7,000円」のように、グループごとに異なる個別価格を設定できます。

**Q3. 掛け率と個別価格が両方設定されている場合、どちらが優先されますか？**

A. **商品の個別価格が優先**されます。グループへの掛け率はあくまで「個別設定がない商品へのデフォルト価格」として機能します。

**Q4. グループに属していない会員（グループ未設定の会員）はどうなりますか？**

A. グループに属していない会員には、通常の定価が表示されます。グループ価格はグループに紐づいているため、グループ未設定の会員には影響しません。

**Q5. 価格変更はリアルタイムで反映されますか？**

A. はい、管理画面で設定を保存した直後からサイトに反映されます。ただし、ページのキャッシュが有効になっている場合は、キャッシュが切れるまで反映されない場合があります。キャッシュをクリアすると即座に反映されます。

## この章のまとめ

この章では、価格管理アドオンを使ってグループごとの価格設定を行う方法を解説しました。

- **掛け率**とは定価に対するパーセンテージで、例えば80%なら定価の80%の価格になります
- 会員グループに掛け率を設定すると、そのグループの会員全員に自動的に計算された価格が表示されます
- 特定の商品には個別価格を設定でき、グループの掛け率より優先されます
- 端数処理（切り捨て・切り上げ・四捨五入）も設定できます
- 設定した価格は商品一覧からカート・注文確認まで一貫して適用されます

### 次章への橋渡し

価格の設定ができたら、次は**支払い方法の制限**です。法人取引では銀行振込や後払いが主流ですが、一般会員向けのクレジットカード決済をそのまま法人会員にも見せてしまうと、現場で混乱が生じることがあります。

次の第5章では、会員グループごとに利用できる支払い方法を制限する「支払い方法管理アドオン」の設定方法を解説します。
