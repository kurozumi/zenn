## この章で学ぶこと

前章では、BtoB ECサイトに必要な要件を整理しました。この章ではいよいよ実装の第一歩として、**会員グループ管理プラグインの導入と基本設定**を行います。

この章を読み終えると、次のことができるようになります。

- EC-CUBEオーナーズストアからプラグインをインストールできる
- 会員グループを作成して会員を割り当てられる
- 商品やカテゴリに閲覧制限をかけられる

「プラグインって難しそう」と感じている方も大丈夫です。管理画面の操作だけで設定が完結しますので、コードを書く必要はありません。順を追って丁寧に説明します。

---

## 会員グループ管理プラグインとは

会員グループ管理プラグインは、EC-CUBEの会員を**グループ単位で管理する**ためのプラグインです。

EC-CUBEの標準機能では、すべての会員に同じ条件でサービスを提供します。しかしBtoBサイトでは、「法人会員には卸価格を見せる」「一般消費者には法人向けの商品を非表示にする」といった使い分けが必要です。

このプラグインを導入すると、以下のことが可能になります。

| できること | 具体例 |
|---|---|
| 会員をグループに分類する | 「法人会員」「個人会員」「VIP会員」など |
| グループごとに商品を見せる・隠す | 法人会員だけが業務用商品を閲覧できる |
| グループごとにカテゴリを制限する | 「卸価格商品」カテゴリは法人会員のみ |
| 後述のアドオンと組み合わせる | 卸価格・支払い方法・配送方法の制限 |

このプラグイン単体でも十分な価値がありますが、第3章以降で紹介するアドオン群と組み合わせることで、より強力なBtoBサイトが構築できます。

- プラグインURL: https://www.ec-cube.net/products/detail.php?product_id=2439

## EC-CUBEオーナーズストアとは

プラグインのインストール手順の前に、**EC-CUBEオーナーズストア**について説明します。

EC-CUBEオーナーズストアは、EC-CUBE専用のプラグインやテンプレートを販売・配布しているマーケットプレイスです。WordPressのプラグインストアに相当するものだと考えてもらえばわかりやすいでしょう。

オーナーズストアの主な特徴は以下のとおりです。

- EC-CUBEの開発元（株式会社イーシーキューブ）が運営している公式ストア
- 無料・有料のプラグインがそろっている
- EC-CUBEの管理画面から直接インストール・アップデートできる
- プラグインの品質審査が行われているため安心して導入できる

会員グループ管理プラグインは有料プラグインです。購入にはオーナーズストアへの登録が必要です。

---

## インストール手順

### 1. オーナーズストアへの登録と購入

まず、[オーナーズストア](https://www.ec-cube.net/)でアカウントを作成し、会員グループ管理プラグインを購入します。

購入が完了すると、ライセンスキーが発行されます。このライセンスキーはインストール時に必要になりますので、メモしておきましょう。

### 2. EC-CUBE管理画面との連携設定

EC-CUBEの管理画面からオーナーズストアに接続するために、認証が必要です。

1. EC-CUBE管理画面にログインする
2. 左メニューから **「オーナーズストア」→「認証キー設定」** を選択する
3. オーナーズストアで発行した認証キーを入力して保存する

これにより、管理画面からプラグインを直接インストールできるようになります。

### 3. プラグインのインストール

認証設定が完了したら、プラグインをインストールします。

1. 管理画面の **「オーナーズストア」→「プラグイン」** を選択する
2. 購入済みプラグインの一覧から「会員グループ管理プラグイン」を探す
3. **「インストール」** ボタンをクリックする
4. インストール完了のメッセージが表示されるまで待つ（数秒〜数十秒程度）

:::message
インストール中は画面を閉じないでください。処理が中断される場合があります。
:::

### 4. プラグインの有効化

インストールが完了しただけでは、プラグインはまだ動作しません。有効化の操作が必要です。

1. 管理画面の **「オーナーズストア」→「プラグイン」** を選択する
2. インストール済みプラグインの一覧から「会員グループ管理プラグイン」を探す
3. ステータスが「無効」になっているので、**「有効化」** ボタンをクリックする
4. ステータスが「有効」に変わればインストール完了です

有効化が完了すると、管理画面の **「会員管理」** メニューに「会員グループ一覧」の項目が追加されます。

## グループの作成方法

プラグインが有効化されると、会員グループを自由に作成できます。ここでは最も基本的な「法人会員」グループを作成する例で説明します。

### 管理画面での操作手順

1. 管理画面にログインする
2. 左メニューから **「会員管理」→「会員グループ一覧」** を選択する
3. 左上の **「新規登録」** ボタンをクリックする
4. 次の項目を入力する

| 項目 | 入力例 | 説明 |
|---|---|---|
| 会員グループ名 | 法人会員 | フロント側に表示されるグループ名 |
| 管理名 | 法人会員（管理用） | 管理画面のみで使う名前。空欄のままでも構いません |

5. **「保存」** ボタンをクリックする

### 複数グループの設計例

実際のBtoBサイトでは、ビジネスの要件に合わせて複数のグループを作成します。

| 会員グループ名 | 用途 |
|---|---|
| 法人会員 | 法人契約済みの取引先 |
| 代理店会員 | 販売代理店向け |
| VIP法人会員 | 取引額の大きい優良顧客 |
| 一般会員 | 審査前または個人のお客様 |

グループ設計はサイト設計の根幹になります。最初から複雑にしすぎず、最小限のグループからスタートして、必要に応じて追加していくことをおすすめします。

## 会員をグループに割り当てる方法

グループを作成したら、会員を割り当てます。割り当て方法は2種類あります。

### 個別に割り当てる方法

1名ずつ手動で割り当てる場合は次の手順で操作します。

1. 管理画面の **「会員管理」→「会員一覧」** を選択する
2. 対象の会員を検索して選択する
3. 会員詳細画面を開く
4. **「会員グループ」** の項目を探す（プラグイン導入後に追加された項目です）
5. **「会員グループ」** のチェックボックスから、割り当てたいグループにチェックを入れる（複数選択可）
6. **「登録」** ボタンをクリックする

1人の会員に複数のグループを同時に割り当てることも可能です。少数の会員を管理する段階では、この手動割り当てで十分対応できます。

### CSVで一括割り当てする方法

会員数が多い場合は、CSVファイルを使って一括で割り当てることができます。

**CSVファイルの準備**

まず、現在の会員データをCSVでエクスポートします。

1. 管理画面の **「会員管理」→「会員一覧」** を選択する
2. **「CSVダウンロード」** ボタンをクリックする
3. ダウンロードしたCSVをExcelまたはGoogleスプレッドシートで開く

エクスポートされたCSVに、会員グループの列（`group_code`など）が追加されています。各会員のグループコードを入力します。

```csv
id,email,name01,name02,group_code
1,yamada@example.com,山田,太郎,corporate
2,suzuki@example.com,鈴木,花子,general
3,tanaka@example.com,田中,次郎,corporate
```

**CSVのインポート**

1. 管理画面の **「会員管理」→「会員グループ一覧」** を選択する
2. **「CSVインポート」** ボタンをクリックする
3. 作成したCSVファイルを選択してアップロードする
4. インポート完了のメッセージが表示されれば成功です

:::message
CSVインポートは既存の会員データを上書きします。インポート前にバックアップを取得することを強くおすすめします。
:::

## 商品・カテゴリに閲覧制限をかける方法

会員グループを設定したら、次は商品やカテゴリの閲覧制限を設定します。これにより「法人会員だけが見られる商品」を作ることができます。

### 商品編集画面での設定

1. 管理画面の **「商品管理」→「商品一覧」** を選択する
2. 閲覧制限をかけたい商品を選択して編集画面を開く
3. **「公開設定」** または **「表示制限」** セクションを探す（プラグイン導入後に追加された項目です）
4. **「閲覧可能な会員グループ」** の項目で「法人会員」にチェックを入れる
5. **「更新」** ボタンをクリックして保存する

この設定をすると、未ログインの訪問者や「法人会員」以外のグループに属する会員は、その商品を閲覧できなくなります。

### カテゴリ編集画面での設定

商品1点ずつに制限をかけるのは手間がかかります。商品の数が多い場合は、**カテゴリごとに制限をかける**方法が便利です。

1. 管理画面の **「商品管理」→「カテゴリ一覧」** を選択する
2. 制限をかけたいカテゴリを選択して編集画面を開く
3. **「閲覧可能な会員グループ」** の項目で「法人会員」にチェックを入れる
4. **「更新」** ボタンをクリックして保存する

カテゴリに制限をかけると、そのカテゴリに属するすべての商品が自動的に制限対象になります。「業務用商品」カテゴリに制限をかければ、そのカテゴリ内のすべての商品が法人会員専用になるというわけです。

### 制限の動作確認

設定後は、必ず動作確認を行いましょう。

- ブラウザを別窓でシークレットモードで開き、未ログイン状態でアクセスする
- 制限をかけた商品・カテゴリが表示されないことを確認する
- 法人会員のアカウントでログインして、正しく表示されることを確認する

## Option1モードとは何か

会員グループ管理プラグインには**Option1モード**という設定があります。

Option1モードは、プラグインの動作を拡張するための設定です。具体的には、後述するアドオンプラグイン（卸価格・承認制など）を使う際に必要となる機能を有効にします。

### Option1モードを使う場合

次のアドオンを導入する予定がある場合は、Option1モードを有効にしてください。

- 承認制アドオン（第3章）
- 卸価格・割引価格アドオン（第4章）
- 支払い方法・配送方法の制限アドオン（第5・6章）

### Option1モードを使わない場合

会員グループ管理プラグインの基本機能（グループ分け・閲覧制限）だけを使う場合は、Option1モードを有効にする必要はありません。

### Option1モードの設定方法

1. 管理画面の **「オーナーズストア」→「プラグイン設定」** を選択する
2. 会員グループ管理プラグインの設定画面を開く
3. **「Option1モード」** のスイッチをオンにする
4. **「保存」** ボタンをクリックする

:::message
Option1モードはアドオン導入前にあらかじめ有効にしておくことをおすすめします。後から有効にした場合、既存のグループ設定に影響が出ることがあります。
:::
