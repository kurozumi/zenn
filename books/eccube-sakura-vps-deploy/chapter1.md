## VPSとは

VPS（Virtual Private Server）は、1台の物理サーバーを仮想的に分割した専用環境です。共有レンタルサーバーと比べて自由度が高く、EC-CUBEのような本格的なECサイトの運用に適しています。

| | 共有レンタルサーバー | VPS |
|---|---|---|
| 価格 | 安い（月500円〜） | やや高い（月643円〜） |
| 自由度 | 低い（設定変更不可） | 高い（何でも設定できる） |
| 性能 | 他ユーザーと共有 | 割り当てリソースが保証される |
| 管理難度 | 簡単 | やや難しい |

EC-CUBEは本格的なECサイト向けのシステムのため、VPSを推奨します。

## さくらVPSのプラン選び

[さくらのVPS](https://vps.sakura.ad.jp/) には複数のプランがあります。EC-CUBEの本番環境には **2GB以上のメモリ** を推奨します。

| 月額（年払い） | メモリ | CPU | SSD | 推奨用途 |
|---|---|---|---|---|
| 590円〜 | 512MB | 1コア | 25GB | EC-CUBEには非推奨 |
| 807円〜 | 1GB | 2コア | 50GB | 開発・テスト用 |
| 1,594円〜 | 2GB | 3コア | 100GB | 小規模本番（〜月間1万PV） |
| 3,227円〜 | 4GB | 4コア | 200GB | 中規模本番（〜月間10万PV） |
| 6,454円〜 | 8GB | 6コア | 400GB | 大規模本番 |

※ 料金はリージョン（大阪・東京・石狩）によって異なります。年払い時の最安値を記載しています。

:::message
最安プランの512MBはEC-CUBEには非推奨です。composerの実行時にメモリ不足になります。本番環境は**2GB以上**を選択してください。
:::

## 契約手順

1. [さくらのVPS](https://vps.sakura.ad.jp/) にアクセス
2. 希望のプランを選択して「お申し込み」
3. OSに **Ubuntu 24.04 LTS** を選択
4. rootパスワードを設定（安全なパスワードを使用すること）
5. 申し込み完了後、メールでIPアドレスが通知される

## 初回SSH接続の確認

契約完了後、ローカルマシンのターミナルからSSH接続できることを確認します。

```bash
ssh root@<サーバーのIPアドレス>
```

初回接続時は契約時に設定した **root パスワード** の入力を求められます。パスワードを入力してログインしてください。

```
root@<サーバーのIPアドレス>'s password: （パスワードを入力）
```

「Welcome to Ubuntu 24.04 LTS」のようなメッセージが表示されれば接続成功です。

:::message
次章でSSH鍵認証を設定すると、パスワード入力なしで接続できるようになります。また root でのログインも禁止してセキュリティを高めます。
:::

:::message alert
さくらVPSの **パケットフィルター機能** がデフォルトで有効になっている場合があります。有効になっているとOS側のufw設定が干渉する場合があるため、コントロールパネルから無効化しておくことを推奨します。
:::

## ドメインの準備

EC-CUBEをHTTPS（SSL）で公開するにはドメインが必要です。お名前.comやムームードメインなどでドメインを取得し、取得したドメインのDNS設定でVPSのIPアドレスを指定してください。

DNS設定の反映には数分〜数時間かかる場合があります。次章以降の作業を進めながら待ちましょう。
