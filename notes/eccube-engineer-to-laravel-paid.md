## 優先して覚えるべきこと

| 優先度 | 学習項目 | 理由 |
|---|---|---|
| 高 | Eloquent + with() | アプリ全体に関わる基礎 |
| 高 | Bladeテンプレート | 毎日使う |
| 高 | ルーティング + Route::resource() | REST API作成に必須 |
| 中 | マイグレーション | DB管理の基本 |
| 中 | FormRequestバリデーション | セキュリティ上重要 |
| 中 | サービスプロバイダー | DIコンテナの理解 |
| 低 | Livewire / Inertia | 案件による |

Symfonyの知識があれば、**2〜3週間で業務レベルのコードが書ける**ようになります。まずEloquentとBladeを集中的に学ぶことをおすすめします。
