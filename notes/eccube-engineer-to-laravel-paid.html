ディレクトリ構造の比較

EC-CUBE / Symfony          Laravel
────────────────────────   ─────────────────────────
src/Controller/         →  app/Http/Controllers/
src/Entity/             →  app/Models/
src/Repository/         →  （Eloquentに統合）
src/Form/               →  app/Http/Requests/
src/EventListener/      →  app/Listeners/
templates/              →  resources/views/
config/services.yaml    →  app/Providers/
config/routes.yaml      →  routes/web.php
var/log/                →  storage/logs/
var/cache/              →  storage/framework/cache/
public/                 →  public/

優先して覚えるべきこと

優先度 学習項目 理由
高 Eloquent + with() アプリ全体に関わる基礎
高 Bladeテンプレート 毎日使う
高 ルーティング + Route::resource() REST API作成に必須
中 マイグレーション DB管理の基本
中 FormRequestバリデーション セキュリティ上重要
中 サービスプロバイダー DIコンテナの理解
低 Livewire / Inertia 案件による

Symfonyの知識があれば、2〜3週間で業務レベルのコードが書けるようになります。まずEloquentとBladeを集中的に学ぶことをおすすめします。
